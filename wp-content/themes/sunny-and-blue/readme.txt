== Description ==

Sunny and Blue is a mobile-friendly(responsive) WordPress theme for publishing a WebComic with the use of the Comic Easel plugin.  Developed using Bootstrap 3(with the ThemeBlog theme) that makes it mobile and tablets friendly.

== Installation ==

You can install Sunny and Blue from the **Themes > Add New** page in the administrative dashboard. Just do a search for `Sunny and Blue` and the first result should be the one you're looking for.

= Manual Installation =

1. Download and extract `Sunny and Blue` from the [WordPress.org theme directory](https://wordpress.org/themes/sunny-and-blue) (for the most recent stable release).
2. Upload the `Sunny and Blue` directory to your `wp-content/themes` directory.
3. Activate `Sunny and Blue` through the **Appearance > Themes** page in the administrative dashboard.
